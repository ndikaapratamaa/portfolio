About Me

Halo! Saya Andika Pratama, seorang pelajar dan pengembang web yang bersemangat untuk menciptakan tampilan digital yang menarik, responsif, dan bermakna. Saya mulai tertarik pada dunia pemrograman sejak duduk di bangku sekolah menengah, dan sejak saat itu saya terus mengasah kemampuan saya dalam HTML, CSS, dan JavaScript untuk membangun pengalaman web yang interaktif. Saya juga gemar mempelajari Python untuk backend dan logika program, serta menggunakan framework modern seperti React untuk menciptakan tampilan yang dinamis. Tujuan saya sederhana — membuat website yang tidak hanya terlihat bagus, tapi juga memberikan pengalaman yang menyenangkan bagi pengguna. Di luar dunia coding, saya senang mengeksplorasi desain UI/UX, membaca artikel teknologi, dan mengerjakan proyek-proyek kecil yang menantang kemampuan saya. Saya percaya bahwa setiap baris kode adalah langkah kecil menuju hasil besar yang berarti.
